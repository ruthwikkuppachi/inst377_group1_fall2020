Documentation


Libraries:
The libraries we used to create this application were express, sqlite3, and node.js. Express and node.js were used to create the server that the application runs on. From node.js the command npm install was used to install all the dependancies. The command npm start-watch is used to start the server. Sqlite3 was used to create a user database for the sign in function of the application. The database stores the user's username and password.
