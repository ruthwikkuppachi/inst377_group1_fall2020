Help


On the "Home" page of this website, the user can filter for transactions involving PG County by the Zip Code that the transaction was made in, the Payee Name of the transaction, and the agency name of the transaction. After submitting a search, it displays the total amount (in USD) that was involved in transactions affecting the particular agency, payee, or zip code.


On the "About" page, the user can find information about the website and its purpose. They can also find information about the team members behind this website.


On the "Documentation" page, users can find information about the technologies used for this website and any commands that developers need to know to work with the website in the future.


On the sign-in page, users can login to their accounts on this website. They can create a new account by clicking on "Create a New Account" or update their password using the "Forgot Password?" option. To sign-in, users need their IDs and their password. Incorrect passwords will not let them "login". If correct password is entered for a specific ID, it displays a "Welcome" message indicating that the "login" was successful. A new account can't be created if there exists another account with the same username. An error message ("ERROR") will be displayed if a username already exists.
